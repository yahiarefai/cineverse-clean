# CineVerse - Movie Booking Demo

A simple responsive Movie Booking Website built with HTML, Tailwind CSS, and JavaScript.

How to run:

1. Open `index.html` in your browser (no build step required).

2. Click "Book Now" to go to the seat selection page.

Files:
- `index.html` - Landing page with hero and movies grid
- `seats.html` - Seat selection layout and price summary
- `styles.css` - Small custom CSS
- `script.js` - Seat generation and booking logic

Notes:
- Uses CDN Tailwind for styling.
- Prices and occupied seats are sample data inside `script.js`.
